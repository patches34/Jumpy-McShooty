# Jumpy-McShooty
Made By: `Nodadic Sage Games`

## Game Identity
An endless run and gun shooter where the player can run on any surface.

## Design Pillars

- Agile
- Overwhelming
- Skilled

## Summary

### Genre

- 2D Platformer

### Mechanics

- Platforming
- Twin Stick Shooting
- Endless Level Generation
- Scoring System
